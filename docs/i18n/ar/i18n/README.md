  العوائق, إلغاء المهام)

- **مَشهد اللاعب**: فحص التحويلات قبل أن يتم الحصول عليها

المفاتيح ( **التحليلات – االتيبيتكات** )

- المرحيبات، الهندسة الجبرية، الجغرافيا، توليد الكود
- التحUILTماء (مطابق للJSON)، الترجمة، توليد الإشارات التوضيحية
- رفض الأمان (المحتوى المضر)، التعداد، المنطق البولين

#### لماذا یفشل تحامل علي Antigravity / Gemini CLI علي السيرفر الناقل؟

يوفر مزودي **تحامل علي Antigravity** ومؤالفهم **Gemini CLI** هذام Google OAuth
2.0**. يتطلب هذام Google تطابق `عنوان التحويل` في تدفق التحامل مع واحدة من معامل
التسجيل المتقدمة مع هذام Google Cloud Console.

يستضيف هذام OmniRoute معامل التسجيل **لإتصال ب `localhost` فقط**. عند الوصول الي
OmniRoute علي السيرفر الناقل (مثل `https://omniroute.myserver.com`), يرفض هذام
Google التحامل مع:

```
خطأ 400: عدم تطابق عنوان التحويل
```

#### ال решение: تحديد معامل تسجيلك الخاص

تحتاج إلى إستكتاب هذام **تعامل OAuth 2.0ID** مع هذام Google Cloud Console مع
عنوان سيرفرك.

#### إجراءات متتالية

**1. افتح هذام Google Cloud Console**

اذهب الي: [https://console.cloud.google.com/apis/credentials ]

**2. أعمل هذام تعامل OAuth 2.0 جديدي**

- انقر علي **"+"- لإضافة كredentialsجديدي** → **"هذام تعامل OAuth"**
- أنواع التطبيق: **"تطبيق عبر الويب"**
- الإسم: أي شيء (مثل `عبر OmniRoute السيرفر الناقل`)

**3. إضف URL التجزئة المصرح بها**

في حقول **"التجزئة المصرح بها**" ، أضف:

```
https://[X8X]your-server.com/X24X]/callback
```

> استبدل `your-server.com/X24X ] بنطاق سيرفرك أو عنوان IP الخاص بيك (إنما يحتاج
> للرغم بيك إذا ايرقة `http://45.33.32.156:20128/callback`).

**4. Salvaremos التوقعات والحفظ للبيانات الشخصية**

بعد إنشاءه ، سوف يظهر **clientId** و **clientSecret** من قِبل Google.

**5. تعيين متغيرات البيئة**

`https://your-server.com/callback` وسوف تعمل عملية التشخيص الإلكتروني.

#### طريقة إجتياز مؤقتة (بدون تهيئة تعريفات خاصة)

إذا كنت لا تريد تهيئة تعريفات خاصة الآن، لا مانع من استخدام مسار **URL manual**:

1. سيفتح OmniRoute URL تحريرية Google
2. بعد ما تدعو من قبلك إلى مصادقة، سيحاول Google تحويلك إلى `localhost` (الذي
   سيفشل في الخادم البعيد)
3. **صقه الرابط الكامل** من العنوان البارز
4. أدخل الرابط في الحقل الظاهر في النافذة المحمولة
5. اختر **"اربط"**

> هذا العقد المحيط يعمل بشكل سليم، لأن رموز المصادقة في الربط تم إصابتها ويربطها
> بشكل سليم بغض النظر عمّا إذا تحققت النعم الأخرة أو لا.
